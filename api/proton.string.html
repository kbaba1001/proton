proton.string

String utilities.

prune

(prune s)(prune s len)(prune s len side)(prune s len side dots)

Returns pruned string that is shortened to the certain length and added dots. The leaving side can be selected from :left, :right, or :both.

split-at

(split-at s x)

Returns a vector of substrings split at the position. You can supply an integer or list as the splitting position.
